# Real-Time Crypto Analytics Dashboard 🚀

This project is a live cryptocurrency tracker built as part of the Navyan Program. It fetches real-time data from the CoinGecko API and provides a localized dashboard for users.

## Key Features
- **Live Data Fetching:** Tracks the top 20 cryptocurrencies in real-time.
- **Dual Currency Support:** Displays prices in both **INR (₹)** and **USD ($)**.
- **Market Alerts:** Includes visual indicators (🟢 UP / 🔻 DOWN) based on 24h price changes.
- **Auto-Refresh:** The dashboard updates every 60 seconds automatically.

## Tools Used
- **Language:** Python
- **Libraries:** Pandas, Requests, Google Colab Data Tables
- **API:** CoinGecko API
